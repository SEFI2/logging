# api-server
## Install
- Install MongoDB
- Install Node

## Npm install

```
npm install
```

## Start MongoDB
```
mongo --port 2121 --dbpath ~/data/db

```

## Start API Server 
```
npm start
```

## Start tests
```
npm test
``` 

## Change logs path
- Go to .env

## Explanation
- Tail reads newly added lines only after its execution
